Princípais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> ,essa tag é NÃO SEMÂNTICA/ SÓ A FORMA

Nesta frase, tenho um termo em destaque usando a tag <strong>, essa é SEMNTICA

Itálico / Ênfase

Nessa frase, temos um termo em itálico usando a tag <i> (NÃO SEMÂNTICA/FORMA);

Nesta frase, temos um termo em ênfase usando a tag <em>, (SEMÂNTICA)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse paragrafo. No qual o <big> ta ficando fora de uso e o <small> ainda ta funcionando, para melhor informações perquisar nos links que o guanabara deixou na no git

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto com inserido para dar ênfase e indicar que ele foi adicionado depois.

Existe também o <u> (NÃO SEMÂNTICA) que deixa o texto sublinhado.

Texto Sobrescrito

Usamos para inserir coisas do tipo (2 pequeno) x20+3, usamos o <sup>

Texto Subscrito

Usamos para inserir coisas (2 pequeno) como H2O, com <sub>
